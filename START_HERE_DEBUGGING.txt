🚀 HOW TO USE

STEP 1: Start server
  $ python manage.py runserver
  (Keep this terminal open to see logs)

STEP 2: Send a test message
  Option A - From browser:
    Open chatbot UI and type a message

  Option B - From terminal:
    curl -X POST http://localhost:8000/api/chat/ \
      -H "Content-Type: application/json" \
      -d '{"message": "Show me diamonds under $5000"}'

STEP 3: Watch the debug output
  Look at Django terminal for:
    [CHAT START] - Request begins
    [PHASE N] - Each phase with detailed logs
    ✓ Success indicators
    ✗ Error indicators
    [API] ▶ Returning HTTP 200 - Request complete

📋 DEBUGGING CHECKLIST

When something breaks:

1️⃣ Check HTTP status
   HTTP 200 = Good, go to #2
   HTTP 500 = Error, check [API ERROR]

2️⃣ Check tool selection
   → Tools showing = Good, go to #3
   ✗ No tools = Check keywords in message

3️⃣ Check tool execution
   [TOOL] ✓ Result length = Good, go to #4
   [TOOL] ✗ ERROR = Database/parameter issue

4️⃣ Check Gemini API
   [GEMINI] ✓ Response received = Good, go to #5
   [GEMINI] ✗ FATAL ERROR = API key issue

5️⃣ Check response extraction
   [PHASE 5] ✓ Extracted text = Success! ✅
   [PHASE 5] ✗ Empty response = Content parsing issue
